# IronAI Fitness

## Overview
An AI-powered personal trainer that generates personalized workouts, tracks your progress with progressive overload, and gamifies your fitness journey.

## How to Run
1. Open terminal in this folder.
2. Run `npm install` to get dependencies.
3. Run `npm run dev` to start the local server.

## Installation (PWA)
Open the app in your browser (Chrome/Edge) and click the "Install" icon in the address bar to install it as a standalone app on your device.
